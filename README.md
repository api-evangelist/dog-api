# Dog API (dog-api)

The internet's biggest collection of open-source dog pictures. The Dog API (dog.ceo) serves over 20,000 dog images accessible by more than 120 breeds via a free, no-auth REST API that returns JSON.

## APIs

### Dog API

The Dog API (dog.ceo) is the internet's largest open-source collection of dog imagery, exposing 20,000+ photos across 120+ breeds. All endpoints are GET-only and require no authentication.

**Human URL:** [https://dog.ceo/dog-api/](https://dog.ceo/dog-api/)

**Base URL:** `https://dog.ceo/api`
